## 一、目标

用 **可运行的 A 股龙虎榜 + 大模型总结** 小项目，在 GitHub / Gitee 获取搜索流量与 clone，通过：

- `config` 默认 `LLM_BASE_URL = https://www.qinghong.tech/v1`
- README **极简薅羊毛风**：只放注册 / 文档 / 定价三链接，不写具体赠送额度

把开发者导到 **晴红API（AI 中转）** 注册、充最低单、长期调用。

**不引流到「晴红量化数据 API」内置 Key**（避免数据被白薅）。

## 二、双平台 SEO 策略（已拍板）

### 2.1 分工原则

| 平台 | 受众 | 模型关键词（SEO 主词） | 场景词 |
|------|------|------------------------|--------|
| **GitHub** | 国外 / 英文检索 / 海归开发者 | **Claude · GPT · Gemini** | stock, quant, LHB, OpenAI-compatible |
| **Gitee** | 国内开发者 | **智谱 GLM · 千问 Qwen** 为主 | DeepSeek 候补 · **龙虎榜 · 量化 · A股** |

同一套代码，**两个远程仓库、两个仓库名、两份 README 侧重点不同**（见 §四）。

### 2.2 模型与站点真实可用名（qinghong.tech）

写 README / config 时只用站里有的 id（带横杠）：

| 用途 | model id |
|------|----------|
| Gitee 默认示例 | `glm-5.2` |
| Gitee 备选 | `qwen3.7-max` |
| 国内代码向 | `kimi-k2.7-code` |
| DeepSeek 候补 | `deepseek-v4-pro` |
| GitHub 默认示例 | `claude-opus-4-8` 或 `gpt-5.5` |
| GitHub 备选 | `gemini-3.5-flash` |

SEO 正文可同时写别名：`glm5.2`、`GPT-5.5`、`Claude Opus`、`Gemini 3.5`（无横杠便于搜索）。

### 2.3 国内默认模型顺序（config / 示例）

1. `glm-5.2`（智谱生态、Cursor 国产向）  
2. `qwen3.7-max`（千问）  
3. `kimi-k2.7-code`（写代码）  
4. `deepseek-v4-pro`（低价候补，SEO 保留即可）

GitHub 示例默认改为 `gpt-5.5` 或 `claude-opus-4-8`，文案强调 **one OpenAI-compatible endpoint for Claude/GPT/Gemini**.

## 四、README 策略（薅羊毛极简 · 分平台）

### 4.1 共同部分（两站一致）

- **能跑**：`pip install -r requirements.txt` + `python analyze_lhb.py`
- **三链接**：注册 · Apifox 文档 · 模型广场定价

## 五、数据从哪来（禁止白薅量化 Key）

| 方案 | 是否采用 | 说明 |
|------|----------|------|
| 调 Mac「晴红量化数据 API」+ 内置 Key | ❌ **禁止** | 等于送 399 年卡数据 |
| Tushare `limit_list_ths` + 内置 token | ❌ | 需积分权限；token 易泄露 |
| **akshare 拉龙虎榜（东财 → 新浪回退）** | ✅ **采用** | 免费、无 Key；逻辑参考 `~/Desktop/晴红量化数据API/sync/jobs/lhb_fetch.py` |

**实现：** 在本项目 `analyze_lhb.py` 内嵌约 30～50 行 akshare 调用（或精简复制 normalize 逻辑），**不** import 量化 API 的 FastAPI 客户端。

**二次转化（可选，README 一行）：**  
「日更清洗行情/龙虎榜 REST API」→ 私域/飞书（晴红量化数据 API 商业产品），与开源 demo **分开**。

## 六、技术骨架（Done）

```
qinghong-open-traffic/
├── 方案与思路.md           ← 本文件
├── README.md               ← GitHub 首页（Claude/GPT/Gemini SEO）
├── README.gitee.md         ← Gitee 首页（GLM/千问/龙虎榜 SEO）
├── config.example.yaml     ← LLM_BASE_URL / MODEL / API_KEY
├── analyze_lhb.py          ← akshare 龙虎榜 + OpenAI SDK → qinghong.tech
├── requirements.txt        # akshare, openai, pyyaml, pandas
└── LICENSE                 # MIT
```

**依赖：** 用户自填 `LLM_API_KEY`（晴红控制台创建），仓库内 **零密钥**。

**默认 config 分平台建议：**

```yaml
# GitHub 示例 config.example.yaml
LLM_BASE_URL: "https://www.qinghong.tech/v1"
LLM_MODEL: "gpt-5.5"
LLM_API_KEY: "your-key-here"

# Gitee 示例（README 内展示即可，或 config.example.zh.yaml）
LLM_MODEL: "glm-5.2"
```
